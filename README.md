# Resto chat backoffice app

Aplicacion gestion restaurant y menu y llamados

## Tecnologias

* Node 18+
* React
* Material UI
* Firebase
* Fromik


## Ejecutar la aplicaion

```bash
#!/bin/bash

nvm use 18

# env
FIREBASE_API_KEY=""
FIREBASE_AUTH_DOMAIN=""
FIREBASE_PROJECT_ID=""
FIREBASE_STORAGE_BUCKET=""
FIREBASE_MESSAGING_SENDER_ID=""
FIREBASE_APP_ID=""
FIREBASE_MEASUREMENT_ID=""

npm run start
```
